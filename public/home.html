// DOCUMENTAÇÃO
RoForger Studio
IDE visual para criação de scripts Roblox. Builder de canvas infinito, sistemas prontos e gerador de Lua.
1. Criando um projeto
Clique em "⚡ Novo Projeto", dê um nome, escolha um tipo e um ícone. O projeto fica salvo na nuvem automaticamente.
2. Usando o Builder
Arraste sistemas do Explorer para o canvas. Conecte blocos, configure propriedades e gere o script Lua completo.
3. Sistemas prontos
80+ sistemas: movimentação, combate, DataStore, UI, admin e mais. Clique em "Copiar Código" para usar direto no Roblox Studio.
